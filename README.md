### 🎵 Sonora — Serverless Music Streaming Platform

My MSc dissertation project exploring the implementation of a variable-bitrate music-streaming service using a serverless AWS architecture.

The system is designed to process uploaded audio into multiple AAC bitrate representations, package tracks for MPEG-DASH playback and distribute them through a content delivery network.

- Event-driven audio processing using AWS Lambda
- Multi-bitrate AAC encoding using FFmpeg
- MPEG-DASH packaging and adaptive playback
- Cloud-based storage and delivery
- Performance, scalability, cost and quality evaluation

**Technologies:** React, TypeScript, AWS Lambda, S3, CloudFront, DynamoDB, FFmpeg, MPEG-DASH
